The paint function takes an App and a color, and paints the app's window with the specified color. The paint function is sensitive to the app's paintbrush size and will paint neighboring pixels of the mouse position, so that the proper diameter of brushstroke is applied.

Parameters
minipaintthe App to paint upon
colorthe color to be painted with
radiusthe radius of the brushstroke
m_xthe x-value of the central pixel for this paint action
m_ythe y-value of the central pixel for this paint action
Returns
std::map<std::pair<int, int>, sf::Color> a map of the affected pixel colors prior to paint action
